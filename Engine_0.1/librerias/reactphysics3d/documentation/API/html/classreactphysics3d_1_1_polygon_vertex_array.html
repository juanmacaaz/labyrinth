Detailed Description

This class is used to describe the vertices and faces of a polyhedron mesh.

A PolygonVertexArray represents an array of vertices and polygon faces of a polyhedron mesh. When you create a PolygonVertexArray, no data is copied into the array. It only stores pointer to the data. The purpose is to allow the user to share vertices data between the physics engine and the rendering part. Therefore, make sure that the data pointed by a PolygonVertexArray remains valid during the PolygonVertexArray life.
