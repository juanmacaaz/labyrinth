Detailed Description

This class is a data-structure contains the pairs of bodies that are overlapping during the broad-phase collision detection.

This class implements the pair manager described by Pierre Terdiman in www.codercorner.com/SAP.pdf.

Add a pair of bodies in the pair manager and returns a pointer to that pair.
